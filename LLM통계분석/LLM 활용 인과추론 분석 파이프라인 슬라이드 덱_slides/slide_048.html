웹에서 자료를 가져오는 3가지 방법

방법 적용 상황 도구 주의점
공식 API KOSIS, 공공데이터포털, FRED, World Bank처럼 API가 제공되는 경우 `requests`, 인증키, 파라미터 기반 요청 호출 제한(rate limit), 키 관리, 쿼리 파라미터 검증 필요
정적 페이지 스크래핑 HTML에 표 데이터가 직접 포함된 페이지 `requests` + `BeautifulSoup` + `pandas.read_html` 페이지 구조 변경 시 selector가 깨질 수 있음
동적 페이지 자동화 자바스크립트 렌더링/버튼 클릭 후 데이터가 보이는 경우 (예: 일부 회의록) `Selenium`(Chrome/Edge WebDriver), 필요 시 Playwright 속도 저하, 로그인/팝업 처리, 법적/윤리적 허용범위 확인

원칙: API 우선 → 정적 스크래핑 → 동적 자동화(Selenium) 순으로 선택
