연구 논문 단계별 프롬프트 생성기

연구 단계를 고른 뒤 8가지 구성요소(역할·맥락·과업·입력·출력·제약·기준·Few-shot)를 채웁니다. 칸을 비워 두면 해당 단계의 표준 예시 문구가 자동으로 반영됩니다. 프롬프트 생성 후 API로 실행하세요.

지금은 파일(file://)로 열려 있습니다. 이 상태에서는 127.0.0.1 로의 요청이 브라우저에서 막혀 Failed to fetch 가 자주 납니다.
  1. 터미널에서 폴더로 이동한 뒤 python openai-local-proxy.py 실행
  2. 브라우저에서 http://127.0.0.1:8788/ 로 이동 (프록시가 떠 있어야 합니다)
  3. 페이지가 뜨면 API 베이스 URL 칸을 비워 둔 채 API로 실행
올바른 사용법: openai-local-proxy.py를 실행한 뒤, HTML 파일을 더블클릭하지 말고 브라우저에서 http://127.0.0.1:8788/ 로 접속하세요. 서버가 같은 주소에서 HTML과 API를 함께 제공하므로 CORS 없이 동작합니다. 터미널: python openai-local-proxy.py
직접 OpenAI에 붙이려면 API 베이스에 https://api.openai.com 를 넣을 수 있으나, 일반 브라우저에서는 여전히 차단됩니다. 키는 로컬에서만 사용하세요.
